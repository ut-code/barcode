# 機能パターン

**機能パターン**は QR コードの位置や向き、白黒のマスの大きさなどを判別するための部分です。そのためこの部分は、どのようなデータを入れるかにかかわらず、QR コードには必ず含まれます。この部分があることで、QR コードを正しく読み取ることができるようになります。

では、ボタンを押して、機能パターンを挿入してみましょう。
