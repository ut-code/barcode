# 機能パターン

**機能パターン**はQRコードの位置や向き、白黒のマスの大きさなどを判別するための部分です。そのためこの部分は、どのようなデータを入れるかにかかわらず、QRコードには必ず含まれます。この部分があることで、QRコードを正しく読み取ることができるようになります。

では、「機能パターンを挿入」ボタンを押して、機能パターンを挿入してみましょう。

<FunctionalPatternPlayground />

機能パターンを挿入できたら「次へ」を押して次に進みましょう。
