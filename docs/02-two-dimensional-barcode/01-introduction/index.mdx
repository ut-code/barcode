# QRコードの概要

**QRコード**は、二次元コードの一種です。二次元コードは、バーコードのような一次元コードとは異なり、縦横に並んだデータを読み取ることができます。そのため、一次元コードよりも多くの情報を記録することができます。

今回扱うQRコードは、大きく分けて次の3つの部分に分かれます。

- 機能パターン: QRコードの位置や向き・マスの大きさを特定するためのパターン
- データ部分: 伝えたい情報が記録された部分。
- 形式情報: QRコードのマスの数や誤り訂正の度合いなどの情報
