# データの挿入

「符号化」と「誤り訂正」の章で作成したデータを、QRコードに挿入していきましょう。

示された場所に、順番に従ってデータを入力します。0 は白 (そのまま) で、1 は黒にします。

「入力をスキップ」ボタンを押すと自動で入力することもできます。
