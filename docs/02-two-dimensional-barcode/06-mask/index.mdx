# マスク

**マスク**は、白や黒のマスだけが連続してしまうことや、機能パターンに似たパターンが出現してしまうことを防ぐために、QR コードに適用される処理です。

具体的には、指定された位置のマスを反転します。反転するパターンを複数用意しておき、その中からマスの連続や機能パターンに似たパターンの出現が少ないパターンを選択します。
