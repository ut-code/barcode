# マスク

**マスク**は、白や黒のマスだけが連続してしまうことや、機能パターンに似たパターンが出現してしまうことを防ぐために、QRコードに適用される処理です。

具体的には、指定された位置のマスの白・黒を反転します。反転するパターンを複数用意しておき、その中からマスの連続や機能パターンに似たパターンの出現が少ないパターンを選択します。

ここでは、簡単のため1つのパターンのみ用意しています。下の図で色のついた部分のマスが反転されます。

![マスクの例](./mask.drawio.svg)

「マスクをかける」ボタンを押してみましょう。
