# 符号化

QRコードでは、伝えたい情報を白と黒のマスに変換して表現します。この白と黒のマスはそれぞれ数字の0と1に対応しています。そのため、アルファベットや日本語などの様々な文字をQRコードで扱うためにはそれらの文字を0と1に変換する必要があります。この操作を**符号化**といいます。

QRコードの符号化にはいくつかのモードがあり、それぞれのモードによって扱える文字の種類や文字数が異なります。

このプレイグラウンドでは「英数字モード」と「8ビットバイトモード」を用意しています。アルファベット・数字・一部の記号で構成されたメッセージの場合は「英数字モード」を、日本語を含む場合は「8ビットバイトモード」を選択してください。

では、符号化モードを選択し、伝えたいメッセージを入力して「コードを生成」ボタンを押しましょう。

:::info

英数字モードで使用できる記号は、半角スペース, `$`, `%`, `*`, `+`, `-`, `.`, `/`, `:`です。

:::
