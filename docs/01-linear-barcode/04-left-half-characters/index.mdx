# 左側のキャラクタ

次に左側のキャラクタを作ってみましょう。

左側のキャラクタには、2つの表 (ナンバーセットAとナンバーセットB) があります。この表のどちらかを使うことで数字を表すことができます。例えば、ナンバーセットAを使って「0」を表すには、「白白白黒黒白黒」とします。ナンバーセットBを使って「0」を表すには、「白黒白白黒黒黒」とします。表が2つある理由については、次の章で説明します。

| 数字 | ナンバーセットAのパターン |
| ---- | ------------------------- |
| 0    | 白白白黒黒白黒            |
| 1    | 白白黒黒白白黒            |
| 2    | 白白黒白白黒黒            |
| 3    | 白黒黒黒黒白黒            |
| 4    | 白黒白白白黒黒            |
| 5    | 白黒黒白白白黒            |
| 6    | 白黒白黒黒黒黒            |
| 7    | 白黒黒黒白黒黒            |
| 8    | 白黒黒白黒黒黒            |
| 9    | 白白白黒白黒黒            |

| 数字 | ナンバーセットBのパターン |
| ---- | ------------------------- |
| 0    | 白黒白白黒黒黒            |
| 1    | 白黒黒白白黒黒            |
| 2    | 白白黒黒白黒黒            |
| 3    | 白黒白白白白黒            |
| 4    | 白白黒黒黒白黒            |
| 5    | 白黒黒黒白白黒            |
| 6    | 白白白白黒白黒            |
| 7    | 白白黒白白白黒            |
| 8    | 白白白黒白白黒            |
| 9    | 白白黒白黒黒黒            |

こちらも7つのバーでできています。右側のキャラクタでは「黒白黒白」の順番になっていましたが、左側のキャラクタでは「白黒白黒」の順番になっていますね。

左側のキャラクタの時と同じようにマスをクリックすることで、パターンを変えることができます。好きな数字を表すパターンを作ってみてください。
