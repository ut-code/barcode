# Leading Digit

先ほどの説明の通り「左側のキャラクタ」にはどういう訳か、AとBの２種類の表し方がありました。

なぜ２種類も表し方があったかというと、この「リーディングディジット」のためです。

リーディングディジットは左側の６つのキャラクタの種類に依存して、自動的に決定されます。

例えば、日本国内で使われる「リーディングディジット」は「４」ですが、\
４の場合、キャラクタの並びは次のようになります。
「 A B A A B B 」

//リーディングディジットの表を入れたいです。
}
