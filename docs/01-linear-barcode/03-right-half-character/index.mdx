# 右側のキャラクタを読もう。

０～９までの数字を線で表したものを「キャラクタ」と言います。

「キャラクタ」は左側の数字で２つ、右側の数字で１つの種類があるので、計３つの表し方があります。

１種類しかないので右側のキャラクタから見てみましょう。

１０種類ありますが、どれも全長は７マス分で「黒白黒白」となっていますね。\
ちなみに、これら右側のキャラクタの表し方を「 number set C 」といいます。

{
//set Cの表を入れたいです
}

下のキャラクタをクリックすることで好きな数字を作成できます。
