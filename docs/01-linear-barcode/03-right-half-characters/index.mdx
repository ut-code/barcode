# 右側のキャラクタ

まずは、右側のキャラクタを作ってみましょう。

基本的に1つのキャラクタが1つの数字を表します。

右側のキャラクタは、次のような表 (ナンバーセットC) に基づいて決まっています。例えば、「0」という数字を表すには「黒黒黒白白黒白」とします。簡単ですね。実は、左側のキャラクタは右側よりも少し複雑になので、次の章で扱います。

| 数字 | パターン       |
| ---- | -------------- |
| 0    | 黒黒黒白白黒白 |
| 1    | 黒黒白白黒黒白 |
| 2    | 黒黒白黒黒白白 |
| 3    | 黒白白白白黒白 |
| 4    | 黒白黒黒黒白白 |
| 5    | 黒白白黒黒黒白 |
| 6    | 黒白黒白白白白 |
| 7    | 黒白白白黒白白 |
| 8    | 黒白白黒白白白 |
| 9    | 黒黒黒白黒白白 |

どれも7つのバーでできていて、「黒白黒白」の順番になっていますね。

では、実際にどのようなパターンがどのような数字を表すのか試してみましょう。

次のマスをクリックすることで、パターンを変えることができます。好きな数字を表すパターンを作ってみてください。
