# 構成

一次元コードの基本構成を見てみましょう。

複数のパーツで成り立っており、それぞれを大まかに解説すると、

#### ・ 「レフト」,「センター」,「ライト」 ガードバー

バーコード全体の初めと終わりを明示し、左と右を仕切ります

#### ・ 「左側」,「右側」 データキャラクタ

左側は６つの、右側は５つの数字に対応してます

#### ・ チェックデジット

終わりに配置され、バーコードが正しく読めてるか確認してくれます

#### ・ リーディングディジット

初めに配置され、左側の線の配置によって自動的に決定されます

となります。
