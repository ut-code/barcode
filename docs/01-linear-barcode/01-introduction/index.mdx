# 一次元コードとは。

バーコードとも呼ばれるこの「一次元コード」は、その名の通り「線」の太さと配置によってデータを表現できます。

私達がよく目にする下の画像は、一次元コードの中でも13桁の数字を表す「JAN-13」というものです。
