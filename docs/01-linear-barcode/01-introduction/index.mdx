# バーコードとは

次の画像のようなものがバーコードです。みなさん一度は目にしたことがあるのではないでしょうか。

<p style={{ width: "350px" }}>
  {new LinearBarcodeEAN13([4, 5, 6, 9, 9, 5, 1, 1, 1, 6, 1, 7]).render()}
</p>

バーコードは、その名の通り黒と白の線のパターンによってデータを表現します。

製品の識別、在庫管理、販売追跡などさまざまな用途で利用されています。

これから、バーコードの仕組みを解説していきますが、まずは、バーコードが実際にどのようなものなのかを体験してみましょう。

次の入力欄に数字を入力すると、その数字に対応したバーコードが表示されます。後で説明しますが、最後の数字はチェックデジットと呼ばれるもので他の数字をもとに計算されるものなので変えることはできないため、注意してください。
