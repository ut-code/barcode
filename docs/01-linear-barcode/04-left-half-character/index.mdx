# 左側のキャラクタ

左側のキャラクタの表し方は２種類あります。\
「 number set A 」と「number set B」です。

先ほどまでとは異なり、「 number set A , B 」の両方とも、色の順番が「白黒白黒」と反転していますね。\
（先ほどの「 number set C 」は「黒白黒白」でした。）

ですが、A , B , C のいずれも、一つのキャラクタの長さが７マス分であることは変わりません。\
それでは、A と B の違いはどこにあるのでしょうか？

答えは「黒いマスの個数」です。\
黒いマスが奇数個なら A に、偶数個なら B になります。\
（例えば 「Aの０」と「Bの０」を比べてみると、Aは黒マス3個、Bは黒マス4個になっています。）

{
//set AとBの表を入れたいです
}
